##Media Queries

You can use it in your html like this:

```html
<link rel="stylesheet" media="(max-width: 800px)" href="my-style.css" />
```

Or you can use it in your css like this:

```html
@media screen and (min-width: 480px) {
    body {
        background-color: lightgreen;
    }
}
```
